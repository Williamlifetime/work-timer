## 功能特性

本插件具有以下特点：

- 下班倒计时实时展示，下班前提示
- 自定义的称呼
- 加班自动计时
- 底部状态栏信息

## 插件设置

1. Ctrl + Shift + P 打开命令行输入 打工计时器 进入设置菜单
2. 点击右下角状态栏中的计时器，打开设置菜单
